This archive contains matlab files for all the questions in assignment2 of pattern recognition.

1)MFCC_Norm.m is a function used to normalize mfcc feature vector to unit variance and zero mean.
2)getDynamicFeatures function returns delta and deltadeltas (dynamic features) of mfcc coefficients.
3)getPlots.m plots the female and music audio time signals.
4)spectgramPlots.m plots the spectrogram of signals.
5)CorrealtionCalc.m plots the correlation matrix of both spectrogram and cepstrogram coefficients.
